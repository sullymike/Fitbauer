\begin{abstract}
\noindent \textbf{Fitbauer} es una aplicación de escritorio para cargar, doblar,
\emph{simular} y \emph{ajustar} espectros Mössbauer de $^{57}$Fe. La simulación
corresponde al \emph{modelo directo} —singletes, dobletes y sextetes—, mientras
que el ajuste puede ser \emph{discreto} (esos mismos componentes) o mediante
\emph{distribuciones} de campo hiperfino o cuadrupolar, que se obtienen
\textbf{ajustando} los datos (una distribución de probabilidad se infiere del
espectro, no se «simula» arbitrariamente).

\medskip\noindent Fitbauer nació con una idea: hacer algo \textbf{nuevo, fácil de
entender e intuitivo}, con \textbf{simulación en vivo} y con lo necesario para
ajustar espectros Mössbauer sin fricción. Existen otros programas —algunos
comerciales, otros gratuitos o de uso académico— pero no siempre mostraban u
ofrecían lo que se necesitaba o se quería para el día a día; de ahí este
proyecto. La sección~\ref{sec:porque} explica el porqué y la
sección~\ref{sec:comparacion} lo sitúa, de forma neutral, junto a otras
herramientas del campo.

\medskip\noindent El manual sigue el flujo de trabajo real: primero la
\emph{calibración} —que se fija y se reutiliza—, después la carga y visualización
de datos, la simulación y el ajuste de un calibrado, y por último el ajuste por
distribuciones. Los recuadros \textbf{Nota}, \textbf{Importante} y
\textbf{Consejo} resaltan detalles prácticos.
\end{abstract}
