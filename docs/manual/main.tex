\begin{abstract}
\noindent Este manual describe el uso de \textbf{Fitbauer}, la aplicación de
escritorio para cargar, doblar, simular y ajustar espectros Mössbauer de
$^{57}$Fe. Se organiza siguiendo el flujo de trabajo real: primero la
\emph{calibración} —que se fija y se reutiliza—, después la carga y
visualización de datos, la simulación y el ajuste de un calibrado, y por último
la simulación de distribuciones. Los recuadros \textbf{Nota}, \textbf{Importante}
y \textbf{Consejo} resaltan detalles prácticos.
\end{abstract}
