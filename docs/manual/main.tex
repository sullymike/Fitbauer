\begin{abstract}
\noindent \textbf{Fitbauer} es una aplicaciГіn de escritorio para cargar, doblar,
\emph{simular} y \emph{ajustar} espectros MГ¶ssbauer de $^{57}$Fe. La simulaciГіn
corresponde al \emph{modelo directo} вҖ”singletes, dobletes y sextetesвҖ”, mientras
que el ajuste puede ser \emph{discreto} (esos mismos componentes) o mediante
\emph{distribuciones} de campo hiperfino o cuadrupolar, que se obtienen
\textbf{ajustando} los datos (una distribuciГіn de probabilidad se infiere del
espectro, no se В«simulaВ» arbitrariamente).

\medskip\noindent Fitbauer naciГі con una idea: hacer algo \textbf{nuevo, fГЎcil de
entender e intuitivo}, con \textbf{simulaciГіn en vivo} y con lo necesario para
ajustar espectros MГ¶ssbauer sin fricciГіn. Existen otros programas вҖ”algunos
comerciales, otros gratuitos o de uso acadГ©micoвҖ” pero no siempre mostraban u
ofrecГӯan lo que se necesitaba o se querГӯa para el dГӯa a dГӯa; de ahГӯ este
proyecto. La secciГіn~\ref{sec:porque} explica el porquГ© y la
secciГіn~\ref{sec:comparacion} lo sitГәa, de forma neutral, junto a otras
herramientas del campo.

\medskip\noindent El manual sigue el flujo de trabajo real: primero la
\emph{calibraciГіn} вҖ”que se fija y se reutilizaвҖ”, despuГ©s la carga y visualizaciГіn
de datos, la simulaciГіn y el ajuste de un calibrado, y por Гәltimo el ajuste por
distribuciones. Los recuadros \textbf{Nota}, \textbf{Importante} y
\textbf{Consejo} resaltan detalles prГЎcticos.
\end{abstract}
